# Security Policy

## Reporting a Vulnerability

* Do not post vulnerabilities publicly.
* Contact the maintainers immediately at [gmail] with a detailed report.
* Include:
  - Steps to reproduce
  - Potential impact
  - Suggested mitigation if possible

## Supported Versions

* We actively support the latest release of this project. Older versions may not receive security updates.

## Response

* The team will acknowledge your report within 48 hours.
* We will work to provide a fix as soon as possible.
